How to use

Choose File
Upload any image from your device to get started.
Drag handles
Drag the 8 blue handles around the image to extend or shrink the canvas in any direction.
Width / Height
Type exact pixel dimensions to set the total canvas size.
Top / Bottom / Left / Right
Type exact pixel values to set each margin individually. The canvas resizes to fit.
Color
Choose the fill color for the extended area.
Pick
Click 💧 to activate the eyedropper and sample any color from your screen. On Safari/Firefox, tap the color square instead to use the system eyedropper.
Center Image
Equalizes all margins so the original image is centered on the canvas.
Zoom
Adjust the preview zoom level. This does not affect the exported image.
Save Image
Downloads the canvas as a PNG, named with its final dimensions.
